# Doc2Exam - 智能考试系统

一个功能完整的自动考试程序，支持从 Excel、Word、PDF 等文档导入选择题库，自动出题，记录错题等功能。

## 🌐 在线演示

https://jhfzhupr6nxxw.ok.kimi.link

## ✨ 主要功能

### 1. 题库导入
- 支持 **Excel (.xlsx/.xls)**、**Word (.docx/.doc)**、**PDF (.pdf)**、**文本 (.txt/.csv)** 格式
- 拖拽上传，自动解析题目、选项、答案、分类、难度、解析
- 支持批量导入多个文件

### 2. 题库管理
- 查看所有导入的题库
- 按名称和分类搜索
- 预览题库中的题目详情
- 删除不需要的题库

### 3. 考试功能
- 自定义出题数量（1-100题）
- 按分类筛选题目
- 按难度筛选（简单/中等/困难）
- 随机顺序开关
- **错题重练模式** - 只练习之前做错的题目
- 考试计时功能
- 题目标记功能
- 题号导航

### 4. 错题本
- 自动记录答错的题目
- 统计错题总数和错误次数
- 按分类筛选错题
- 查看错题详情和正确答案
- 手动移除已掌握的错题

### 5. 考试记录
- 保存所有考试历史
- 显示得分、用时、答题详情
- 统计平均分、最高分、最低分
- 查看每次考试的详细答题情况

### 6. 数据持久化
- 所有数据保存在浏览器本地存储
- 刷新页面数据不丢失

## 📋 题目格式示例

系统支持自动识别以下格式的选择题：

```
1. 这是题目内容
A. 选项A
B. 选项B
C. 选项C
D. 选项D
答案: A
分类: 数学
难度: 中等
解析: 这是答案解析
```

Excel格式建议按以下列顺序：
| 题目 | 选项A | 选项B | 选项C | 选项D | 答案 | 分类 | 难度 | 解析 |

## 🛠 技术栈

- **React + TypeScript + Vite**
- **Tailwind CSS + shadcn/ui** - 现代化UI组件
- **xlsx** - Excel文件解析
- **mammoth** - Word文件解析
- **react-dropzone** - 文件拖拽上传

## 🚀 本地运行

```bash
# 克隆仓库
git clone https://github.com/wsjq5477/doc2exam.git
cd doc2exam

# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```
